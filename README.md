# MarketPlace_WPF
MarketPlace, written on C# and interoperability with MSSQL, VK, Instagram. Description is inside.

To run the application, you need to create a database from the CreateDB.sql file, and also change the DataSouce in the App.config file and set your own name of the client server. After you will be available the first user with mail: admin@mail.ru and password: admin
